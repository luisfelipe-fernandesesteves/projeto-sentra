# 🌎 Sobre o Projeto

O **Sentra** é um projeto tecnológico voltado para o combate e monitoramento de incêndios florestais na Amazônia.

A proposta utiliza tecnologia, análise de dados e monitoramento inteligente para auxiliar na identificação de focos de incêndio e otimizar ações preventivas.
